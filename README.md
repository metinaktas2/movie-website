# 🎬 Movie Web Application

This project is a modern and responsive movie website where users can explore popular films, search for specific titles, and view detailed information about each movie. It is built using **HTML, CSS, and JavaScript** with a clean and user-friendly interface.

---

## ⚙️ Features

🎥 Display popular movies on the homepage
🔍 Search functionality (by movie name or genre)
🖱️ Hover effect to view movie details (overview, rating, etc.)
📱 Fully responsive design (mobile, tablet, desktop)
⭐ Movie ratings with dynamic color indicators

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* TMDb API (for movie data)

---

## 🚀 How to Use

1. Clone or download the project
2. Open the project folder
3. Go to `script.js`
4. Replace the API key:

```js
const API_KEY = "YOUR_API_KEY";
```

5. Open `index.html` in your browser
